# Portfolio Website (Vite + React + Tailwind)

This is a starter portfolio site built with Vite, React, TypeScript and Tailwind CSS. It includes a Glassmorphism dark theme with accent color #EE4C2C.

Quick start

1. Install dependencies

```bash
cd website
npm install
```

2. Run dev server

```bash
npm run dev
```

Build

```bash
npm run build
npm run preview
```
